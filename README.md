# Tinder Wealth

> A gamified **personal finance & investment platform** that lets users swipe through investment opportunities, build portfolios, and track wealth growth — making finance engaging and accessible.

🌐 **Live Demo:** [Tinder-Wealth on GitHub Pages](https://pinnacleroute.github.io/Tinder-Wealth)

---

## ✨ Features

- 💊 Swipe-style investment discovery
- 💼 Portfolio builder & tracker
- 📈 Wealth growth analytics
- 🎯 Financial goal setting
- 🔔 Market opportunity alerts
- 📱 Fully responsive UI

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML5, CSS3, JavaScript |
| Deployment | GitHub Pages |
